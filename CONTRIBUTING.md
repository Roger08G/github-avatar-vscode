# Contribuir

## Configuración local

```bash
bun install
bun run check
```

Usa `F5` en VS Code para ejecutar la extensión en un Extension Development Host.

## Pull requests

- Mantén los cambios acotados.
- Añade o actualiza pruebas cuando cambie el comportamiento.
- Ejecuta `bun run format` antes de hacer commit.
- No registres tokens de acceso ni incluyas datos reales de cuentas en fixtures.
- Documenta los cambios visibles para el usuario en `CHANGELOG.md`.
