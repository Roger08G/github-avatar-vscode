# Historial de cambios

Todos los cambios relevantes de este proyecto se documentarán en este archivo.

## [0.1.0] - 2026-06-16

### Añadido

- Autenticación de GitHub a través del proveedor integrado de VS Code.
- Tarjeta de perfil con avatar y estadísticas públicas de la cuenta.
- Entrada de la cuenta en la barra de estado.
- Respaldo del perfil en caché.
- Flujo de desarrollo basado en Bun.
- CI multiplataforma y releases etiquetados de GitHub.
